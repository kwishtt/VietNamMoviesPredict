# Tuần 8 — Cập nhật Web App & Feature Importance Analysis (Phase 2)

## Mục tiêu
Tích hợp model Pre-Release mới vào Web App và phân tích Feature Importance.

## Việc cần làm

### 1. Cập nhật Backend (`prediction_service.py`)

- [ ] Load model mới `pre_release_rf_model.pkl`
- [ ] Cập nhật `prepare_features()` - chỉ dùng pre-release features
- [ ] Loại bỏ logic tính ROI, roi_vs_vote
- [ ] Cập nhật `_calculate_dynamic_probability()` với logic mới
- [ ] Test API endpoint `/predict`

### 2. Cập nhật Frontend (`index.html`)

- [ ] **Loại bỏ** input fields:
  - ❌ Revenue (Doanh thu)
  - ❌ Vote Average (Điểm đánh giá)
  
- [ ] **Thêm** input field:
  - ✅ Release Date (Ngày phát hành dự kiến)
  
- [ ] Cập nhật wording:
  - "Dự Đoán" → giữ nguyên (vì giờ là dự đoán thật)
  - Subtitle: "Nhập thông tin phim **trước khi phát hành** để dự đoán khả năng thành công"

### 3. Cập nhật JavaScript (`app.js`)

- [ ] Cập nhật form handling (không gửi revenue, vote)
- [ ] Thêm Date picker cho release_date
- [ ] Cập nhật hiển thị Feature Importance

### 4. Feature Importance Analysis

- [ ] Phân tích Top 10 features quan trọng nhất của model mới
- [ ] So sánh với model cũ
- [ ] Tạo biểu đồ minh họa
