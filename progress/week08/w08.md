# Tuần 8 — Cập nhật Web App

## Mục tiêu
Tích hợp model Pre-Release mới vào Web App.

## Việc cần làm

### 1. Cập nhật Backend (`prediction_service.py`)

- [ ] Load model mới `pre_release_rf_model.pkl`
- [ ] Cập nhật `prepare_features()` - chỉ dùng pre-release features
- [ ] Loại bỏ logic tính ROI từ revenue
- [ ] Cập nhật `_calculate_dynamic_probability()` với logic mới
- [ ] Test API endpoint `/predict`

### 2. Cập nhật Frontend (`index.html`)

- [ ] **Loại bỏ** input fields:
  - ❌ Revenue (Doanh thu)
  - ❌ Vote Average (Điểm đánh giá)
  
- [ ] **Giữ/Thêm** input fields:
  - ✅ Budget (Ngân sách)
  - ✅ Runtime (Thời lượng)
  - ✅ Genres (Thể loại)
  - ✅ Release Month (Tháng phát hành) - optional
  
- [ ] Cập nhật wording: Subtitle mô tả rõ "dự đoán trước phát hành"

### 3. Cập nhật JavaScript (`app.js`)

- [ ] Cập nhật form handling (không gửi revenue, vote)
- [ ] Cập nhật hiển thị Feature Importance mới

### 4. Test End-to-End

- [ ] Test với 3-5 cases mẫu
- [ ] Verify kết quả hợp lý

## Giao nộp
- Web App hoạt động với model mới
- Screenshots test cases
