# Tuần 10 — Hoàn thiện, Tổng duyệt & Nộp Final

## Mục tiêu
Hoàn tất báo cáo, slide và mã nguồn; tổng duyệt thuyết trình và nộp sản phẩm cuối cùng.

## Việc cần làm

### 1. Tổng duyệt thuyết trình

- [ ] Practice thuyết trình (15-20 phút)
- [ ] Chuẩn bị trả lời câu hỏi giám khảo:
  - "Tại sao accuracy chỉ 70-80%?" → Giải thích Pre-Release vs Post-Release
  - "Data leakage là gì?" → Sử dụng thông tin tương lai để dự đoán
  - "Tại sao không dùng model 99%?" → Không có giá trị thực tế cho dự đoán
- [ ] Chỉnh sửa theo góp ý giảng viên (nếu có)

### 2. Đóng gói mã nguồn

- [ ] Tạo `requirements.txt` đầy đủ
- [ ] Viết hướng dẫn chạy lại (SETUP.md hoặc trong README)
- [ ] Đảm bảo code chạy được trên máy khác
- [ ] Clean up code thừa, comments không cần thiết

### 3. Chuẩn bị Demo

- [ ] Test Web App hoạt động ổn định
- [ ] Chuẩn bị 3-5 test cases mẫu:
  - Phim budget cao → dự đoán kết quả
  - Phim budget thấp → dự đoán kết quả
  - Phim mùa hè vs mùa xuân → so sánh
- [ ] Backup demo video (phòng trường hợp technical issue)

### 4. Nộp sản phẩm cuối cùng

- [ ] Báo cáo final (Word/PDF)
- [ ] Slide final (PowerPoint/PDF)
- [ ] Mã nguồn (GitHub hoặc zip)
- [ ] Dataset đã xử lý
- [ ] Model files (.pkl)
