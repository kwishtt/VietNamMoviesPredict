# Tuần 10 — Hoàn thiện & Nộp Final

## Mục tiêu
Tổng duyệt, fix bugs và nộp sản phẩm cuối cùng.

## Việc cần làm

### 1. Tổng duyệt thuyết trình

- [ ] Practice 15-20 phút
- [ ] Chuẩn bị trả lời câu hỏi:
  - "Tại sao accuracy 70-80%?" → Giải thích Pre-Release vs Post-Release
  - "Data leakage là gì?" → Sử dụng thông tin tương lai
  - "Feature quan trọng nhất?" → Budget, genres, release timing

### 2. Đóng gói mã nguồn

- [ ] `requirements.txt` đầy đủ
- [ ] Hướng dẫn chạy trong README
- [ ] Clean code thừa

### 3. Demo

- [ ] Test Web App ổn định
- [ ] Chuẩn bị 3-5 test cases
- [ ] Backup demo video (phòng lỗi)

### 4. Nộp sản phẩm

| Item | Format |
|------|--------|
| Báo cáo | .docx/.pdf |
| Slide | .pptx/.pdf |
| Source code | GitHub/zip |
| Model files | .pkl |
